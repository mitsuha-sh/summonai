# /summonai-executor-config — Executor Configuration Wizard

Generate or update `.summonai/executors.toml` interactively.

## Instructions

Follow these steps in order.

### Step 1 — Check for existing config

Use the Read tool to check if `.summonai/executors.toml` exists in the project root.

If the file **exists**, use AskUserQuestion to ask:

> `.summonai/executors.toml` already exists. Overwrite it?
>
> - yes — overwrite with new config
> - no — abort

If the user answers "no" (or any variant meaning cancel), stop and report "Aborted. Existing config was not changed."

### Step 2 — Ask which executor pattern to use

Use AskUserQuestion to present the four patterns:

> Which executor configuration pattern do you want?
>
> **P1** — Claude only (Haiku → Sonnet → Opus, cheapest-first routing)
> **P2** — Codex Plus only (gpt-5.3-codex, single model, all bloom levels)
> **P3** — Codex Pro only (gpt-5.3-codex for bloom ≤3, gpt-5.3-codex-spark for bloom 4-6)
> **P4** — Mixed (Claude Haiku for low, Codex Plus for mid, Claude Sonnet/Opus for high)
>
> Enter P1, P2, P3, or P4:

Accept case-insensitive input (p1, P1, 1 all mean P1).

### Step 3 — Generate the TOML content

Use the exact templates below. Do **not** alter model names, field names, or `{model}` placeholders.

---

#### P1 — Claude only

```toml
# summonai executor config — Pattern P1: Claude only
# Generated by /summonai-executor-config
# Do not commit this file (.summonai/ is git-ignored).

[[capability_tiers]]
executor = "claude"
model = "claude-haiku-4-5-20251001"
max_bloom = 3
cost_group = "low"

[[capability_tiers]]
executor = "claude"
model = "claude-sonnet-4-6"
max_bloom = 5
cost_group = "medium"

[[capability_tiers]]
executor = "claude"
model = "claude-opus-4-7"
max_bloom = 6
cost_group = "high"

[runners.claude]
template = "claude --model {model} --dangerously-skip-permissions"

[defaults]
bloom_level = 3
executor = "claude"
```

## TOML Schema Reference

```
[[capability_tiers]]        # one entry per tier (array of tables)
executor   = "claude"       # CLI tool name matched by runners.*
model      = "..."          # passed to {model} placeholder
max_bloom  = 3              # 1=Remember … 6=Create
cost_group = "low"          # low / medium / high (informational)

[runners.<executor>]
template = "<cmd> --model {model}"   # {model} is replaced at runtime

[defaults]
bloom_level = 3             # default when task omits bloom_level
executor    = "claude"      # optional: pin executor
```
